# Zephyr 🌤️

Next.js weather app!

[🚀 Check it out!](https://zephyr-weather.vercel.app)

## 🖧 Stack

- Next.js for server-side rendering
- OpenWeatherMap API for weather data
- shadcn/ui for components
- TypeScript
- Tailwind CSS
